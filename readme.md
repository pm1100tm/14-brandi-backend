# 프로젝트 개요

### 1. 프로젝트 설명
여성 온라인 쇼핑몰 [브랜디](https://www.brandi.co.kr/) 의 서비스/어드민 기능 구현 프로젝트.
* 서비스 - 실제 유저가 웹사이트에 들어와 상품을 보고 구매를 진행하는 등의 기능.
* 어드민 - 관리자 기능으로써 회원관리(유저, 셀러, 관리자)와 상품 관리 및 결제 완료 후의 과정에 대한 기능.
